Smart IoT Traffic System

Project Objective

  • Detect the congestion on road
  • sending the less congestion road to cars drivers
  • help for solving the problem congestion and less the probability of accidents

Project Components

  1. Node MCU (esp8266 )
  2. Image sensor (camera)
  3. Position detection System

Project Description

  • Then camera take pictures for road and deliver it to Node MCU
  • Node processing the images and send information about the congestion quantity to cloud
  • Then the cloud system compare the data from different traffic signs and send the less congestion and best road for cars drivers
